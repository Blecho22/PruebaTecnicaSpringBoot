# CRUD Vuelos (Spring Boot)

Aplicación con Spring Boot(Java 17, Maven, Jar) que expone un CRUD en memoria para gestionar vuelos con filtros y ordenación.

## Tecnologías
- Java 17
- Spring Boot(Spring Web, Validation)
- Lombok

## Requisitos
- JDK 17
- Maven 3.9+
- Plugin de Lombok

## Ejecutar
**Desde terminal**
